Place bundled `ffmpeg` and `ffprobe` binaries here.

Expected default locations:

- `bin/ffmpeg/windows-x64/ffmpeg.exe`
- `bin/ffmpeg/windows-x64/ffprobe.exe`
- `bin/ffmpeg/linux-x64/ffmpeg`
- `bin/ffmpeg/linux-x64/ffprobe`
- `bin/ffmpeg/linux-arm64/ffmpeg`
- `bin/ffmpeg/linux-arm64/ffprobe`
- `bin/ffmpeg/macos-x64/ffmpeg`
- `bin/ffmpeg/macos-x64/ffprobe`
- `bin/ffmpeg/macos-arm64/ffmpeg`
- `bin/ffmpeg/macos-arm64/ffprobe`

You can override these paths with:

- `FFMPEG_PATH`
- `FFPROBE_PATH`
